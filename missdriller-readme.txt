■Miss Driller ver 0.05
2000/11/5

【概要】
Mr.Driller(c)namcoに、どことなく似ているゲームです。
テスト版です。いくつかの機能は動きません。

【動作環境】
画面が800x600、16bit以上で、次のOSを使用しているコンピュータ。
	Linux
	Windows9x
	BeOS
	FreeBSD4.0
	Irix
	MacOS
	Solaris
SDL-1.1.3以降をインストールしている必要があります
また、SDL_mixer-1.0.5以降がインストールされている必要があります（今のところ）

【動作方法】
■コンパイル
makeと打つだけです

■プレイ方法

%./drillでプログラムが開始します（あるいはdrill.exeをダブルクリック）。

ゲームルールはMr.Drillerです。1000M掘ると終了です。

矢印キーで移動、スペースキーで掘ります。
ジョイスティックの場合、レバーを入れると移動し、0ボタンで掘ります。

ネームエントリーはスペースで文字入力、バックスペースで訂正、リターンで決定です。ジョイスティックの場合、0ボタンが入力、1ボタンが訂正となります。決定はキーボードのリターンか、2ボタンを利用してください。

アプリケーション終了ボタンかエスケープキーを押すといつでもゲームを終了します。


■細かい仕様変更など

【フルスクリーン化】
drill.ini内で
#FULLSCREEN		YES
FULLSCREEN		NO
となっているところを
FULLSCREEN		YES
#FULLSCREEN		NO
などと変更すると、フルスクリーン化します。ゲーム終了にはESCを押してください。

【ジョイスティックの有無】

drill.ini内で
JOYSTICK_ENABLED
をNOにするとジョイスティックを読まなくなります。ドライバーがうまく動作せず、ボタンが入りっぱなしになったり、その他不具合がでるようならジョイスティックの使用を中止してください。
